# AI Vocabulary Exposed

### Arvind Yadav – Claude Vocabulary

A simple reference project that explains important **AI thinking and reasoning vocabulary** used by modern AI systems like Claude.

This repository provides a visual HTML page that helps users understand the **language AI models use when describing reasoning, analysis, and problem solving.**

---

## 📚 Purpose

Large language models often use specific reasoning words such as:

* Analyze
* Evaluate
* Infer
* Deduce
* Compare
* Summarize
* Hypothesize

This project exposes those **thinking patterns and vocabulary** so developers and learners can better understand how AI systems reason.
